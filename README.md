# SSh_VLAN-GNS3
Лабораторная работа: Настройка VLAN, маршрутизации между VLAN и удалённого доступа по SSH
## `Цель работы`
- Научиться создавать виртуальные локальные сети (VLAN) на управляемом коммутаторе.
- Освоить технологию «router-on-a-stick» для маршрутизации между VLAN.
- Настроить доступ к сетевым устройствам по протоколу SSH для удалённого управления.
- Изучить вопросы безопасности SSH: выбор алгоритмов шифрования, работа с устаревшими версиями клиентов.

## `Топология сети`
В работе используется следующая схема:
<img width="1919" height="1142" alt="Screenshot_2" src="https://github.com/user-attachments/assets/1267138e-8920-4913-b3e1-8b70c472feac" />

- **R1** – маршрутизатор Cisco 7200 (IOS 15.3). Интерфейс f0/0 подключён к NAT-сети GNS3 (адрес получает по DHCP). Интерфейс f1/0 включен и на нем висят два sub-int f1/0.10 к VLAN 10 и ub-int f1/0.20 к VLAN 20
- **SW** – коммутатор (в GNS3 можно использовать Ethernet switch или маршрутизатор с модулем ESW). Настроены порты доступа (access) для клиентов и trunk-порт для соединения с R1.
- **PC1, PC2** –  имитируют конечные устройства.

## Необходимое оборудование
- GNS3 версии 2.2.56.1.
- GNS3 VM (Ubuntu 20.04) с интерфейсами: первый – бридж (для доступа к физической сети), второй – NAT (используется для выхода в интернет).
- Образ Cisco 7200 (c7200-adventerprisek9-mz.153-3.XB12).
- Коммутатор – встроенный Ethernet switch GNS3.
