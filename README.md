# SSh_VLAN_DHCP_NAT-GNS3
Лабораторная работа: Настройка VLAN, DHCP, подключение к внешней сети через NAT и настройка  удалённого доступа по SSH
## `Цель работы`
- Научиться создавать виртуальные локальные сети (VLAN) коммутаторе.
- Освоить технологию «router-on-a-stick» для маршрутизации между VLAN.
- Настроить доступ к сетевым устройствам по протоколу SSH для удалённого управления.
- Изучить вопросы безопасности SSH: выбор алгоритмов шифрования, работа с устаревшими версиями клиентов.
- Настройка DHCP servera для автоматического назначения ip адресов.
- Изучения технологии NAT.

## `Топология сети`
В работе используется следующая схема:

- **R1** – маршрутизатор Cisco 7200 (IOS 15.3). Интерфейс f0/0 подключён к NAT-сети GNS3 (адрес получает по DHCP). Интерфейс g1/0 включен и на нем висят три sub-int f1/0.10 к VLAN 10 , sub-int f1/0.20 к VLAN 20 , sub-int f1/0.30 к VLAN 30
- **SW1** – core-коммутатор (в GNS3 можно использовать Ethernet switch или маршрутизатор с модулем ESW). Настроены порт доступа (access) для DHCP-server и trunk-порт для соединения с R1, SW2 , SW3.
- **SW2-3** коммутаторы доступа.
- **PC1, PC2, PC3, PC4** –  имитируют конечные устройства.

## Необходимое оборудование
- GNS3 версии 2.2.56.1.
- GNS3 VM (Ubuntu 20.04) с интерфейсами: первый – бридж (для доступа к физической сети), второй – NAT (используется для выхода в интернет).
- Образ Cisco 7200 (c7200-adventerprisek9-mz.153-3.XB12).
- Коммутатор – встроенный Ethernet switch GNS3.

## Возможные проблемы и их решение
Проблема: Клиенты не получают IP-адреса по DHCP
Решение: Проверьте ip helper-address на сабинтерфейсах роутера и доступность DHCP-сервера.

Проблема: Нет доступа в интернет
Решение: Проверьте настройки NAT (ip nat inside/outside), access-list и маршрут по умолчанию.

Проблема: Не удается подключиться по SSH
Решение: Используйте расширенную команду с указанием алгоритмов или настройте SSH-клиент через ~/.ssh/config.

Проблема: Трафик между VLAN не работает
Решение: Проверьте транковые порты, разрешенные VLAN и настройки сабинтерфейсов на роутере.
